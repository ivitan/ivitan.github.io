几个 Npm 常用命令

命令解析
npm -v显示版本，检查npm 是否正确安装
npm install -g express全局安装express模块
npm list列出已安装模块
npm show express显示模块详情
npm update升级当前目录下的项目的所有模块
npm update express升级当前目录下的项目的指定模块
npm update -g express升级全局安装的express模块
npm uninstall express删除指定的模块
